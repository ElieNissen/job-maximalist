JobMAXIMALIST - Demarrage Windows

Ce dossier est un package portable.
Il n'y a rien a installer.

Ordre recommande :

1. Gardez tout le dossier tel quel
2. Double-cliquez sur "1 - Start JobMAXIMALIST.vbs"
3. Attendez pendant le premier lancement

Important
- Le premier lancement peut prendre quelques minutes.
- Gardez Internet actif pendant ce premier lancement.
- L'application peut telecharger son moteur de navigation pour faire fonctionner correctement certaines sources.

Si besoin
- "2 - Repair JobMAXIMALIST.vbs" relance une preparation propre si quelque chose s'est mal passe.
- "3 - Open JobMAXIMALIST data.vbs" ouvre le dossier local ou l'application stocke ses donnees.
