JobMAXIMALIST - Installation macOS

1. Ouvrez "1 - Installer JobMAXIMALIST.pkg".
2. Suivez l'installation.
3. Une fois l'installation terminee, ouvrez "JobMAXIMALIST.app".

Important
- Le premier lancement peut prendre quelques minutes.
- Gardez Internet actif pendant ce premier lancement.
- L'application peut telecharger son moteur de navigation pour faire fonctionner correctement certaines sources.

Ensuite
- "JobMAXIMALIST.app" demarre l'application.
- "Reparer JobMAXIMALIST.command" relance une preparation propre si quelque chose s'est mal passe.
- "Ouvrir les donnees JobMAXIMALIST.command" ouvre le dossier local ou l'application stocke ses donnees.
